# 🏋🏽‍♀️BUILDME💪
「きょうの目標はなんですか？」<br>
1日はじめに目標を立て、1日のおわりに振り返りを行う。<br>
日々自分自身を鍛えるためのサービスです。

# 🏋🏽‍♀️BUILDMEについて
### 登場人物
* 目標を達成したいと望む人
* やってみたいことはあるが行動につながらない人

### ユーザーが抱える課題
* やってみたい・成し遂げたいことがあるが継続できなかったり行動することができない

### 解決方法
* 目標を言葉にすること
* 目標に対して振り返りを実施し、日々改善していくこと

### 望む未来
* 目標を立て、振り返りを行ない、改善できるようになること
* 成し遂げたいことを見いだし、達成に向けて挑戦すること

### サービスをつくった背景
「ダイエットしたいんだけどなかなか続かない」「スキルをとりたいんだけど何から始めたらいいか分からない」と、なにかをやりたいけれど、実際に行動に移せていない友人が身近に多くいます。<br>
具体的な行動に繋がらないそもそもの原因はなんだろう…と考えたところ、なにかを達成するために「なにをするか？」という日々の目標を言語化していないことに原因があると感じました。
自分で目標を立てれて、自分で振り返る、さらにはそれを続けるためのサポートツール付き。そんなアプリがあったらいいなあ！と思い、それをカタチにしたのが「BUILDME」です。

### 機能構成
機能はざっくりと2つに分けられます。
* メイン機能
  * 1日の目標を投稿する
  * 目標に対して振り返った内容を投稿する
* サポート機能：ユーザーがメイン機能を効果的・継続的に使えるようにサポートする
  * ユーザー管理機能（新規登録・ログイン・マイページ)
  * コメント機能
  * 瞑想タイマー
  * メモ書き

# 工夫したポイント
* シンプル・ストレスフリーであること
  * 見た目をシンプルでわかりやすいデザインにした
  * JavaScriptを使い画面遷移をスムーズにすることでストレスを軽減する仕様にした

* 機能の役割を明確に定義することで、どのような目的のサービスか明確にした
  * メインの機能の役割は、ユーザーが「目標を立てる・振り返りをする」ためのサービス
  * サポート機能の役割は、ユーザーがメイン機能を効果的・継続的に使えるようにサポートする
    * マイページ機能：目標を投稿した日にはカレンダーの該当日に●印をつけることで、自分の日々の成果を可視化しモチベーションを高める
    * コメント機能：同じ目標を持つ者同士、コミュニケーションがとれモチベーションを高める
    * 瞑想タイマー：目標を立てる前に瞑想をすることで、心がととのい、明確な目標を立てられるようにサポートする
    * メモ書き機能：モヤモヤした時にメモ書きをすることで、思考が明確になる
